OOP

A programação orientada a objetos é um paradigma de programação, ou seja, um conjunto de conceitos e recursos de linguagem que guiam a forma como programamos.

Tem como peça central os objetos e a forma como eles se comunicam.

Os objetos podem conter dados (atributos) e códigos (métodos)

Outros dois termos importantes na OOP são classes e instâncias

  Classes: são como fábricas de objetos a partir de uma 'forma'

  instâncias: são os objetos fabricados pelas classes
